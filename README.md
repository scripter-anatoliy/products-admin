## Что использовано

- **React 19** + **TypeScript** (strict), **Vite**
- **Zustand** — стейт (авторизация, список товаров, сортировка, поиск, форма добавления)
- **Ant Design** — форма входа, таблица, инпуты, кнопки
- **React Router 7** — маршруты (логин / товары)
- **API:** DummyJSON Auth, DummyJSON Products

## Как запустить

```bash
npm install
npm run dev
```

Открыть в браузере: http://localhost:5173

---

В задании нужны: список товаров с подгрузкой из API, сортировка с сохранением состояния, поиск, добавление и редактирование в таблице, плюс авторизация с «запомнить данные». Данные и UI-состояние (какой столбец сортирован, что введено в поиск, выбранные строки и т.д.) используются в нескольких местах (страница, таблица, форма, хедер). Чтобы не таскать пропсы через много уровней и не дублировать запросы, вынес общее состояние в стейт-менеджер.

**Zustand** выбран как минимальный и достаточный вариант: один store для auth, один для products (список, сортировка, поиск, добавление), без провайдеров и лишнего бойлерплейта. Типизация удобная, при необходимости легко добавить persist (например, для настроек таблицы или «запомнить меня»). Для объёма этого приложения Redux или MobX излишне.
